# Infinite Blackboard

A premium infinite digital blackboard for iPad and desktop, optimized for Apple Pencil.

## Features
- Infinite canvas
- Freehand drawing with smooth curves
- Grid background
- Eraser tool
- Undo/Redo support
- Apple Pencil optimized

## Technologies
- React 18
- Vite
- TypeScript
- HTML Canvas API
- PWA support

## Scripts

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```
